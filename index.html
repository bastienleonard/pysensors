Welcome to pySensors’s documentation!¶

Easily control your hardware sensors (temperature, fan speed, etc.) with a simple Python API.

Go to the project’s page to get the source, report issues and more: https://github.com/bastienleonard/pysensors

The lm_sensors C library¶

This is really a binding for the lm_sensors C library. Its official homepage has been unavailable for a while, but you can still find extensive information from those links:

  • Arch Linux wiki page

  • GitHub project
